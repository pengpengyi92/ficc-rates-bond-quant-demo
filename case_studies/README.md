# Case Studies

This folder contains public-safe fixed-income case studies that connect bond
duration theory with realistic market-move simulations.

Current cases:

- `2026_china_gov_bond_ytd.md`: 2Y / 5Y / 10Y China government bond YTD simulation.
- `2026_30y_rates_trade.md`: 30Y long-duration rates trade simulation.
- `100bn_position_simulation.md`: large-notional duration exposure framework.

These notes use simplified assumptions and approximate public-market yield
moves. They are not real trade records and do not contain proprietary data,
client information, or live trading instructions.
